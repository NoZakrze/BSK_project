Detailed Description

Module responsible for selected a pdf file and signing it with a private key.

Steps that user must perform:

  1. Enter pin code
  2. Select PDF File
  3. Click on "Sign PDF" button When user clicks "Sign PDF" button pdf file will be singed with user's private key
